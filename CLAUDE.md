# easy-extrude (PUBLIC リポジトリ)

## このリポジトリの役割
ここは **公開 (public)** リポジトリ = 無料の「器」。VS Code タイトルバーは **赤**。
緑なら private 側 (`easy-extrude-private`) なので、戦略の話はそちらで行うこと。

## ここで扱ってよいもの
- ブラウザ 3D エディタ / 基本ジオメトリ操作 / JSON エクスポート
- Layout DSL の **公開スキーマ** と **基本コンパイラ** / バリデータ
- サンプル数個 / ローカル単体動作

## ここに置かないもの (private 側 `easy-extrude-private` へ)
- マネタイズ戦略・価格・課金・売上・事業判断
- 判定の **実装**: IK ソルバ / 干渉チェッカ / リーチ計算 / wrench cone / 把持安定性スコア
- 有償テンプレート (完成 DSL)

## 制約は「参照名」まで (実装は持たない)
- DSL には `ik_solvable` / `no_collision` / `within_reach` などの **参照名** は持ってよい。
- それを **解く処理** は書かない。public は宣言・スキーマ・参照名まで。

## AI 向けガード (重要)
このリポジトリで以下を依頼/生成しようとしたら、**作業を中断して警告**すること:
- マネタイズ・価格・戦略に関する記述
- 上記「判定の実装」に当たるコード (IK/干渉/リーチ/安定性の解法)
中断したうえで「それは private 側 (easy-extrude-private) で行ってください」と促す。
